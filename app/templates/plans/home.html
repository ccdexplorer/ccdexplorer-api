The API


The API is authenticated and rate-limited.

Authentication

API v2 is authenticated using a header key x-ccdexplorer-key.
Assuming that your API key is 528e6511-d55a-49d3-a4f1-fcce5eef03cc, then the authentication header you should send will be:

x-ccdexplorer-key: 528e6511-d55a-49d3-a4f1-fcce5eef03cc


Rate Limits

The v2 API endpoints are all rate limited.
